Projects


Nerdy Valentine Amazon Echo App

An Amazon Alexa app that gives the user a nerdy pickup line when asked. It is based on the Space Facts example program and was my first Amazon Alexa App. This won best application at Code Day 2016 and is currently published on the Alexa Skills store.

Mole is a Unit Website

Made as a project for school, this small website helped me to learn how to use a variety of CSS frameworks and further refine my frontend web dev skills. The website explains how one mole of a 24 byte file would take up more than the worlds storage. This was compared to how there were many moles of zinc in a single penny, explaining how small atoms really are. Also, check it out on GitHub.

Arduino Games Platform

A versatile, easy to code and expand game platform which my team and I made 2 games for. People from around the world can build the games platform and record scores on a future web application. The idea of the project was to create an open source game platform in which many different games could be coded on it or added to it. Also, check out the instruction site.

Memory Mashup

This is my first ever Arduino Game, which is basically a two player version of the game simon. The players take turns memorizing a pattern of lights and must repeat it. Also, check out the instruction website.

Positivity

Positivity is a website built for uplifting or positive thoughts. Users go on the site and can post uplifting thoughts which are placed on the main page. Users can then like the most uplifting thoughts and they will rise to the top of the page.

Diopsys, Inc. Website Rebuild

This website was used to give new ideas and show some alternatives to Word Press (calendar and blog features) as well as show responsive website development to the company for their new website which is currently under construction. Also, check it out on GitHub
