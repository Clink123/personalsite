Projects


Mole is a Unit Website

Made as a project for school, this small website helped me to learn how to use a variety of CSS frameworks and further refine my frontend web dev skills. The website explains how one mole of a 24 byte file would take up more than the worlds storage. This was compared to how there were many moles of zinc in a single penny, explaining how small atoms really are. Also, check it out on GitHub.

Nerdy Valentine Amazon Echo App

An Amazon Alexa app that gives the user a nerdy pickup line when asked. It is based on the Space Facts example program and was my first Amazon Alexa App. This won best application at Code Day 2016 and is currently published on the Alexa Skills store.
